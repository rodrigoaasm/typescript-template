## Description

This repository is a template for a Typescript project. This template is already configured with:
 - ESbuild
 - Jest
 - Typescript
 - Eslint

## Install

```sh
npm install
```

## Run Application

#### Run
```sh
npm run start
```

#### Run dev mode
```sh
npm run start:dev
```

#### Build for production
```sh
npm run build
```

## Tests

#### Run unit tests
```sh
npm run test:unit
```

#### Run integration tests
```sh
npm run test:integration
```

#### Run all tests
```sh
npm run test
```
## Build

#### Run production
```sh
npm run start
```
